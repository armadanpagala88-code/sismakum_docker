# Setup Tailwind CSS

## Instalasi

1. Install dependencies:
```bash
cd frontend
npm install
```

2. File yang sudah dibuat:
- `tailwind.config.js` - Konfigurasi Tailwind
- `postcss.config.js` - Konfigurasi PostCSS
- `src/style.css` - Tailwind directives

3. Jalankan development server:
```bash
npm run dev
```

## Komponen yang sudah diupdate dengan Tailwind:

- ✅ Login.vue - Form login dengan gradient background
- ✅ MainLayout.vue - Navigation bar modern
- ✅ Dashboard.vue - Statistik cards dan list

## Custom Classes

Di `src/style.css` sudah ada custom classes:
- `.btn`, `.btn-primary`, `.btn-secondary`, `.btn-success`, `.btn-danger`
- `.form-input`, `.form-textarea`, `.form-label`
- `.badge`, `.badge-draft`, `.badge-diajukan`, dll
