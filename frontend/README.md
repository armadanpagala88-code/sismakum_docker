# Frontend Vue.js - Sistem Pengusulan PERBUB

## Instalasi

1. Install dependencies:
```bash
npm install
```

2. Jalankan development server:
```bash
npm run dev
```

3. Build untuk production:
```bash
npm run build
```

## Fitur

- Login/Logout
- Dashboard dengan statistik
- Daftar pengusulan PERBUB
- Buat pengusulan baru
- Edit pengusulan (draft)
- Detail pengusulan
- Review pengusulan (bagian hukum)
- Upload dokumen pendukung

## Teknologi

- Vue 3
- Vue Router
- Pinia (State Management)
- Axios (HTTP Client)
- Vite (Build Tool)
